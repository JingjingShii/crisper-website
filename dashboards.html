CRISPER Contact Tracing Alerts Web App

This Web App gathers Exposure Site information from multiple State Government sites and presents the locations on a map. It may not include the full detail provided by State Governments, and there may be a lag between the time the data is published by the State Government and when it appears on the map. Some map locations may not be precise due to limited address information or for other reasons. To find the most recent, complete and authoritative data, please visit the appropriate State Government website.

About

The COVID-19 Real-time Information System for Preparedness and Epidemic Response (CRISPER) has been developed to provide accurate and spatially explicit real-time information for COVID-19 cases and contact tracing locations in Australia. The system comprises three main components:

  • Data Engine
  • Data Visualisation and Interactive Mapping Tools
  • Automatic Alert System
This system provides integrated data from multiple sources in one platform which optimises information sharing with public health responders, primary health care practitioners and the general public.
