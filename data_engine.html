About

The data engine is the integration point for the sources of health and geospatial data. It combines data from multiple and often very different public sources into a common format. A suite of carefully tuned data scrapers has been implemented to parse the data formats of the many different public sources.

The data engine is also available to independent developers, including government. The data engine is hosted by the National Computational Infrastructure at the Australian National University.

Privacy

The CRISPER Data Engine can utilise both public data and private data provided by jurisdictional authorities. All private data is managed on secure servers that utilise a new differential privacy capability, RelM implemented as open source by the Software Innovation Institute. Private data is released publicly according to a mathematically rigorous Privacy Budget formulation. Such data is then accessed for use in the CRISPER Dashboards and the CRISPER Alerts through the CRISPER Data Engine.

Usage & Sources

The data used by the Alerts Engine is collected daily from different government sources. It is only as up-to-date and as accurate as the agencies have made available and is dependent on their updates. The types of data collected include, for example, daily covid cases by postcode, daily covid testing numbers by postcode, and contact tracing locations.

  • nsw_postcode_cases - https://data.nsw.gov.au/data/dataset/covid-19-cases-by-location
  • vic_active_cases_postcode - https://www.dhhs.vic.gov.au/victorian-coronavirus-covid-19-data
About

The COVID-19 Real-time Information System for Preparedness and Epidemic Response (CRISPER) has been developed to provide accurate and spatially explicit real-time information for COVID-19 cases and contact tracing locations in Australia. The system comprises three main components:

  • Data Engine
  • Data Visualisation and Interactive Mapping Tools
  • Automatic Alert System
This system provides integrated data from multiple sources in one platform which optimises information sharing with public health responders, primary health care practitioners and the general public.
